# 🐶 Puppy Desert Run

귀여운 강아지가 사막에서 낙타와 선인장을 피해 음식을 먹으며 점수를 얻는 게임입니다.

## 실행 방법
1. `index.html`을 브라우저로 열면 바로 실행됩니다.
2. 스페이스바로 점프하세요!
3. 음식(뼈)을 먹으면 점수가 올라갑니다.
4. 장애물(낙타, 선인장)에 부딪히면 게임 오버!
